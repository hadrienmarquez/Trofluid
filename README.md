# Trofluid
Interface to control srt streaming 

## What is needed to run ?
Only Docker and docker-compose are needed.

## Launching the service
`cd /Path/to/Trofluid`

`docker-compose up`


## Cleaning up routine
If you want to clean up : 

`docker-compose down`
`docker volume prune`
`docker container prune`
`docker image rm Trofluid`
